// Keamanan Berbasis Sistem

Dibangun Aman

Lapisan proteksi terpadu dalam memitigasi celah eksploitasi data sensitif pengguna.

CSRF Protection

Implementasi token per-sesi unik yang diuji ketat di setiap permintaan post melalui modul verify_csrf().

Session Timeout

Kedaluwarsa otomatis sesi pasca 12 jam minim aktivitas. Pembatasan login satu sesi aktif untuk menghindari tumpang tindih akun.

Prepared Statements

Seluruh kueri basis data memanfaatkan parameterized bindings, mengeliminasi risiko peretasan via SQL Injection.

Role-based Access

Restriksi halaman manajemen serta hak akses drive sesuai tingkatan izin pengguna yang bertindak sebagai admin / member.

Activity Logging

Perekaman detail alamat IP, agen peramban, serta kronologi aktivitas demi mitigasi intrusi mencurigakan pada backend.

Session Enforcement

Pemindaian otentikasi ketat secara terpusat di tiap proses request melalui penyertaan berkas auth/auth.php.
