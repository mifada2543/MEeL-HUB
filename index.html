// Keamanan

Dibangun Aman

Lapisan keamanan berlapis untuk melindungi data dan akses sistem.

CSRF Protection

Token per session, divalidasi di setiap POST request via verify_csrf().

Session Timeout

Auto-expire setelah 12 jam inaktif. Single-session enforcement per user aktif.

Prepared Statements

Seluruh query database menggunakan prepared statements mencegah SQL injection.

Role-based Access

Kontrol akses drive dan fitur admin berdasarkan role admin / member.

Activity Logging

Tracking IP, user agent, dan semua aktivitas. Sistem ban IP terintegrasi.

Session Enforcement

Validasi sesi aktif dan role di tiap request melalui auth/auth.php.
